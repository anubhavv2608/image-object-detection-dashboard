# Image Object Detection Dashboard

This project demonstrates object identification using machine learning and a graphical user interface built with Taipy.

## Features

- Upload image
- Detect object using pretrained MobileNetV2 model
- Interactive GUI dashboard

## Technologies

Python  
Taipy GUI  
TensorFlow  
NumPy  
Pillow

## Run Project

Install dependencies:

pip install -r requirements.txt

Run the application:

python app.py
